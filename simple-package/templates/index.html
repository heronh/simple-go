Como criar um projeto em go:
Abra o terminal e navegue até o diretório do seu projeto. Execute o comando go mod init , onde é geralmente o caminho do repositório do seu projeto (ex: github.com/seunome/meuprojeto). Isso criará um arquivo go.mod para gerenciar as dependências do seu projeto.
  • go mod init com.github/teste/simple-package
  • go mod tidy
